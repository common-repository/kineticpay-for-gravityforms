== Installation ==
1. Install & Activate.
2. Add Kineticpay Field on Form From Gravity Form Editor.
2. Update Form.
3. Forms >> Settings >> Kineticpay >> Add New.
4. Insert your API Key.
5. Update Settings.

== Frequently Asked Questions ==

= How to include Transaction ID on payment notification? =

Set the tag {entry:transaction_id} at the event Payment Completion notification. You may refer to [GravityForms Merge Tag](https://docs.gravityforms.com/merge-tags/#entry-data) for more information.
